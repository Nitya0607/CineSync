# CineSync

A full-stack social movie platform built with the MERN stack. Discover movies, manage your watchlist, rate films, and connect with friends.

## Features

- User authentication with JWT
- Browse and search movies
- Rate and review movies
- Personal watchlist management
- Send and manage friend requests
- Admin dashboard for content management

## Tech Stack

**Frontend:** React.js, React Router, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose  
**Auth:** JWT, bcryptjs  

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

```bash
# Clone the repo
git clone https://github.com/Nitya0607/CineSync.git
cd CineSync

# Install backend dependencies
npm install

# Install frontend dependencies
cd frontend
npm install
```

### Running the App

```bash
# Start backend
npm run dev

# Start frontend (open a new terminal)
cd frontend
npm start
```
